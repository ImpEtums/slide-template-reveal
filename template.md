Python代码示例：
```python
def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)
        
# 打印斐波那契数列的前10个数
for i in range(10):
    print(fibonacci(i))
```
